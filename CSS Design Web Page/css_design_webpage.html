Step 1: COLOR!

Color can be specified for web pages by using RGB values, hexadecimal values, HSB values, and color names. In this unit we discussed the case for using HSL as an excellent tool for modifying the lightness/darkness and saturation of a given hue.

Opacity (AKA transparency) can be applied to anything with color. This includes text color, background color, and images.

Use this page and CSS stylesheet to practice with color and opacity values. Use the various resources to change the colors in the text and background of the elements on this page.

  • Apply background color to the container using HSL values.
  • Apply a border, box-shadow, and rounded corners to the container.
  • Use the CSS opacity property to change the opacity of the image.
  • Create a gradient in the background of the container.
  • Create a customized button (button below) that includes a gradient, box-shadow, and border.
  • Use the HSLA specification to change the transparency of a color and/or a drop shadow.
  • Apply text and color styling to the footer area.
  • "I Love Color!" - Create a link to your favorite color picker - style the link.
